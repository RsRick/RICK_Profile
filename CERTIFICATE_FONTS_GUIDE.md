## ğŸ’¡ Best Practices

### DO's âœ…

1. **Use certificate fonts for names only**
   - Recipient names
   - Signature names
   - Special titles

2. **Pair with readable fonts**
   - Use serif fonts for body text
   - Use sans-serif for dates/details
   - Keep it legible

3. **Size appropriately**
   - Names: 36-48px
   - Titles: 24-32px
   - Body: 14-18px

4. **Maintain hierarchy**
   - Largest: Recipient name
   - Medium: Certificate title
   - Smallest: Body text

5. **Test readability**
   - Print preview before finalizing
   - Check at actual size
   - Ensure clarity

### DON'Ts âŒ

1. **Don't use for body text**
   - Certificate fonts are decorative
   - Hard to read in paragraphs
   - Use only for emphasis

2. **Don't mix too many scripts**
   - Maximum 2 certificate fonts
   - Stick to one style
   - Maintain consistency

3. **Don't use all caps**
   - Certificate fonts work best in title case
   - "John Smith" not "JOHN SMITH"
   - Preserve elegance

4. **Don't make it too small**
   - Minimum 24px for names
   - Ensure readability
   - Consider printing

5. **Don't overuse**
   - Less is more
   - Use sparingly
   - Focus on key elements

## ğŸ–¨ï¸ Printing Considerations

### Font Size for Print
- **Names:** 36-48pt (0.5-0.67 inches)
- **Titles:** 24-32pt (0.33-0.44 inches)
- **Body:** 12-16pt (0.17-0.22 inches)

### Resolution
- Use high-quality fonts (Google Fonts are vector-based)
- Print at 300 DPI minimum
- Test print before bulk printing

### Paper Recommendations
- Certificate paper (heavyweight)
- Parchment or linen finish
- Cream or white color
- 24-32 lb weight
