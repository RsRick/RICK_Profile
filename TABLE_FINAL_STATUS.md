## How to Use

### In Editor (Admin)
1. **Insert Table**: Click table icon → Set dimensions → Insert
2. **Edit Cells**: Click any cell and type
3. **Add Row**: Hover over row → Click + button on left
4. **Add Column**: Hover over first row cell → Click + button on top
5. **Settings**: Click ⚙️ button (always visible in top-right)
6. **Format Text**: Select text → Use toolbar (bold, color, fonts, etc.)

### In Popup (Public)
- Tables display correctly
- Content is read-only
- No editing possible
- Professional appearance
