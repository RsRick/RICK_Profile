# Photo Grid - Quick Start Guide

## ðŸš€ How to Use Photo Grids

### Step 1: Open the Editor
1. Go to Admin Panel â†’ Project Management
2. Create or edit a project
3. Scroll to the Rich Text Editor

### Step 2: Insert a Photo Grid
1. Click the **Grid icon** (Grid3x3) in the toolbar
2. A modal will open showing 24 layout options

### Step 3: Select a Layout
1. Browse through the 24 different grid layouts
2. Each shows the number of photos it needs
3. Click on your preferred layout

### Step 4: Add Images
1. Click on each empty cell in the grid
2. Choose upload method:
   - **Image URL**: Paste a direct image link
   - **Upload File**: Select from your computer
3. Click "Add Image" for each cell
4. Repeat until all cells are filled

### Step 5: Insert the Grid
1. Review your grid in the preview
2. Click "Insert Grid" button
3. Grid appears in your editor

### Step 6: Edit or Delete (Optional)
1. Hover over the grid in the editor
2. Click **Settings** icon to edit
3. Click **Delete** icon to remove

## ðŸ“‹ Available Layouts

### Simple Grids (1-5)
- **2 Columns**: 2 photos side by side
- **3 Columns**: 3 photos in a row
- **4 Columns**: 4 photos in a row
- **2x2 Grid**: 4 photos in a square
- **3x2 Grid**: 6 photos in 2 rows

### Asymmetric Layouts (6-10)
- **Large Left + 2 Right**: Featured image on left
- **Large Right + 2 Left**: Featured image on right
- **2 Top + 3 Bottom**: Wide top, 3 small bottom
- **3 Top + 2 Bottom**: 3 small top, wide bottom
- **Large Center + 4 Corners**: Center focus with corners

### Creative Layouts (11-15)
- **Masonry 1**: Tall images on sides
- **Masonry 2**: Tall images alternating
- **Featured + 3**: One large, three small
- **Split + 4**: Two large top, 4 small bottom
- **Showcase**: Alternating sizes

### Gallery Layouts (16-20)
- **Gallery 1**: Tall left, wide right
- **Gallery 2**: Large square with small images
- **Portfolio 1**: Wide top, tall right
- **Portfolio 2**: Mixed sizes
- **Magazine**: Editorial style layout

### Special Layouts (21-24)
- **Hero + Grid**: Hero image with thumbnails
- **Spotlight**: Wide feature with small images
- **Collage 1**: Artistic arrangement
- **Collage 2**: Creative mixed layout

## ðŸ’¡ Tips & Tricks

### Image Quality
- Use high-resolution images (at least 1200px wide)
- Keep file sizes under 2MB for faster loading
- Use JPG for photos, PNG for graphics

### Layout Selection
- Use **2-3 columns** for simple galleries
- Use **Masonry** for varied image sizes
- Use **Featured** layouts for hero images
- Use **Magazine** for editorial content

### Best Practices
- Fill all cells for best appearance
- Use consistent image styles (all photos or all graphics)
- Consider mobile viewing (simpler layouts work better)
- Test different layouts to find the best fit

## ðŸ”§ Troubleshooting

### Grid Not Appearing
- Make sure you clicked "Insert Grid"
- Check that images are loading
- Refresh the editor

### Images Not Loading
- Verify image URLs are correct
- Check file upload completed
- Ensure images are publicly accessible

### Layout Looks Wrong
- Try a different layout
- Check image aspect ratios
- Verify all cells are filled

### Can't Edit Grid
- Hover over the grid to see buttons
- Click the settings icon (gear)
- Make changes and save
