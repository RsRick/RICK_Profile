## üìç Where to Find Font Management

1. Login to your admin panel
2. Click on **"Settings"** in the left sidebar
3. At the top, you'll see a purple card that says **"Font Management"**
4. Click on it to open the font management page

## üé® How to Upload a Custom Font

### Step 1: Get Your Font File
- Download a .ttf font file from a trusted source
- For Bangla fonts, try: [Ekushey Fonts](https://www.omicronlab.com/bangla-fonts.html)
- Popular Bangla fonts: Kalpurush, SolaimanLipi, Nikosh

### Step 2: Upload the Font
1. Go to Font Management page
2. Find the **"Upload Custom Font"** section at the top
3. Enter a name for your font (e.g., "Kalpurush")
4. Click **"Choose File"** and select your .ttf file
5. Click **"Upload Font"**
6. Wait for the success message

### Step 3: Verify Upload
- Your font will appear in the **"Custom Fonts"** section
- It will have a purple **"Custom"** badge
- You'll see a preview with English and Bangla text

## üí° Tips for Choosing Fonts

### For Headings/Titles
- Use **Serif** fonts for elegant, traditional look
  - Playfair Display
  - Merriweather
  - Cinzel

- Use **Display** fonts for bold, modern look
  - Bebas Neue
  - Oswald

### For Body Text
- Use **Sans-serif** fonts for easy reading
  - Open Sans
  - Roboto
  - Lato

### For Bangla Text
- Always use fonts with **"‡¶¨‡¶æ‡¶Ç‡¶≤‡¶æ"** badge
- Noto Sans Bengali is safe and readable
- Test with actual Bangla text before finalizing

### Font Pairing
- Use different fonts for headings and body text
- Example: Playfair Display (heading) + Open Sans (body)
- Don't use more than 2-3 different fonts on your site

## ‚ùì Common Questions

### Q: What file format should my font be?
**A:** Only .ttf (TrueType Font) files are supported.

### Q: How big can my font file be?
**A:** Maximum 5MB per file.

### Q: Can I use the same font everywhere?
**A:** Yes! Select the same font in all sections.

### Q: What if my Bangla text doesn't show correctly?
**A:** Make sure you're using a font with the **"‡¶¨‡¶æ‡¶Ç‡¶≤‡¶æ"** badge. Try Noto Sans Bengali first.

### Q: Can I upload multiple fonts at once?
**A:** Currently, you need to upload one at a time.

### Q: Will my fonts work on mobile phones?
**A:** Yes! Fonts work on all devices.

### Q: Can I undo font changes?
**A:** Yes, just select a different font and save again.

## üö® Troubleshooting

### Problem: Upload button is disabled
**Solution:** 
- Make sure you entered a font name
- Make sure you selected a .ttf file
- Check file size is under 5MB

### Problem: Font doesn't appear after upload
**Solution:**
- Refresh the page
- Check if upload was successful (look for success message)
- Try uploading again

### Problem: Font looks weird on website
**Solution:**
- Make sure the font file is not corrupted
- Try a different font
- Use Google Fonts instead (they're tested and reliable)

### Problem: Can't find a font in the list
**Solution:**
- Use the search box to type the font name
- Check if you're filtering by category
- Click "All" to see all fonts

### Problem: Bangla text shows as boxes
**Solution:**
- Font doesn't support Bangla characters
- Select a font with **"‡¶¨‡¶æ‡¶Ç‡¶≤‡¶æ"** badge
- Try Noto Sans Bengali (always works)
