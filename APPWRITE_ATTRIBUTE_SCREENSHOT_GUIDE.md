## ğŸš¨ Critical Reminders

### âŒ Common Mistakes

1. **Forgetting to check "Array"**
   - Result: Data won't save correctly
   - Fix: Delete attribute and recreate with Array checked

2. **Wrong attribute key**
   - Wrong: `ProjectDetails`, `project_details`, `projectdetails`
   - Right: `projectDetails` (exact case)

3. **Size too small**
   - Wrong: 100, 255
   - Right: 2000 or more

4. **Making it required**
   - Wrong: Required = Yes
   - Right: Required = No

## ğŸ’¡ Pro Tips

1. **Double-check "Array"** - This is the most common mistake
2. **Copy-paste the key** - Use `projectDetails` exactly
3. **Test immediately** - Go to admin panel right after creating
4. **Keep old fields** - Don't delete software, timeframe, etc. for backward compatibility
